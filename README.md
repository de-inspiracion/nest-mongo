## Description

Project to connect mongoDB o cosmosDB for Mongo with  based on DDD architecture

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## ENV
local example

```bash
PORT=3000
CONNECTION_STRING="mongodb://localhost:27017"
```

## Docker Mongo

```bash
docker-compose up
```
